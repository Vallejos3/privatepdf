# PrivatePDF

Free local PDF tools (merge, split, rotate, watermark) in your browser.

## Features

- Merge multiple PDFs
- Split PDF by page range
- Rotate PDF pages
- Add watermark
- Optimize PDF size

**No uploads, 100% local processing**

## Deploy

### Vercel

1. Connect GitHub repo
2. Deploy

### Cloudflare Pages

1. Connect GitHub repo
2. Set build command: `npm run build`
3. Set output dir: `out`
